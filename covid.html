About Project

The novel coronavirus (COVID-19) that was first reported at the end of 2019 has impacted almost every aspect of life as we know it.This SQL data analysis of covid-19 disentangle how covid affected the world.

This exploratory analysis project delves into the global impact of the Covid-19 pandemic on world, using data on death counts, vaccination rates, and other covid-19 related metrics. The purpose of this project is to analyze trends and patterns to gain insights into the impact of the pandemic on the world.
